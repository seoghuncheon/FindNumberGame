# 🎮 숫자 맞추기 게임 (Find Number Game)

C# WinForms를 활용하여 개발한 제한시간 숫자 맞추기 게임입니다

## 💡 주요 기능
* 난이도별(쉬움/보통/어려움) 제한시간 및 기회 차등 부여
* 입력한 숫자에 따른 실시간 UP/DOWN 및 범위 축소 힌트 제공
* SoundPlayer를 활용한 커스텀 효과음 (성공/실패) 장착
* 최소 시도 횟수를 측정하는 최고 기록(Best Score) 갱신 시스템
